AutomataLab

Tool for drawing DFA/NFA and exporting them to PNG.

How to Use:

  • Add State: Click anywhere on the canvas to add a new state
  • Add Transition: Click first state, then second state to add transition (prompts for symbol)
  • Set Initial: Click a state to make it the initial state
  • Toggle Final: Click a state to toggle its final state status
  • Move: Drag states to reposition them
  • Delete: Click a state to remove it and its transitions
  • Export: Click to save the automaton as PNG image

Created for "Computability and Formal Languages"
