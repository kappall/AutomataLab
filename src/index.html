AutomataLab

Tool for drawing DFA/NFA and exporting them to PNG.

How to Use:

  • Add State (A): Click anywhere on the canvas to add a new state
  • Add Transition (T): Click first state, then second state to add transition (prompts for symbol)
  • Set Initial (I): Click a state to make it the initial state
  • Toggle Final (F): Click a state to toggle its final state status
  • Move (M): Drag states to reposition them
  • Delete (D): Click a state to remove it and its transitions
  • Fullscreen (Space): Toggle fullscreen mode for better visualization
  • Export (E): Click to save the automaton as PNG image (cropped to content)

  • Esc: Return to Move mode
